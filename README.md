## Command Line Snyps
to init
    
    // initializing prisma
    npx prisma init --datasource-provider sqlite
    // push the modal to backend
    npx prisma db push
to migrate data 

    npx primsa migrate dev

start client

    npm run client

start server

    npm run server

 start both

    npm run dev
